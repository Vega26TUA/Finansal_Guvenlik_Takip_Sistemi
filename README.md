# Finansal_Guvenlik_Takip_Sistemi
Merkeziyetsiz Finans (DeFi) ekosistemini korumak için tasarlanmış Hibrit Yapay Zeka platformu. 1200+ sentetik veriyle eğitilmiş Makine Öğrenmesi (Random Forest) ile saniyeler içinde zafiyet teşhisi yapar; Google Gemini AI ile kodu analiz edip güvenli sözleşmeyi otomatik yazar. Hızlı, otonom ve güvenilir Web3 siber güvenlik kalkanı.
